To-Do API Starter


A simple, production-ready Node.js API, built with Express.js and Postgres.

/GET /api/todos

Description

Retrieves all To-Do's as an array.

Content Type

application/json

Request Body

none

Sample Response

{
  "message": "ok",
  "todosData": [
    {
      "id": "1",
      "title": "Do something",
      "content": "A description about this task.",
      "is_completed": false
    },
    {
      "id": "2",
      "title": "Another task",
      "content": "Lorem ipsum dolor sit amet."
      "is_completed": true
    },
    {
      "id": "3",
      "title": "A third task",
      "content": "Consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.",
      "is_completed": "false"
    },
  ]
}
/GET /api/todos/{id}

Description

Retrieves a single todo by ID

Content Type

application/json

Request Body

none

Sample Response

{
"message": "ok",
"todo":
  {
    "id": "1",
    "title": "Do something",
    "content": "A description about this task.",
    "is_completed": false
  }
}
/POST /api/todos

Description

Add a new todo to the data.

Content Type

application/json

Request Body

{
  "title": "A new task",
  "content": "Lorem ipsum dolor sit amet.",
  "is_completed": false
}

Sample Response

{
"message": "ok",
"todo":
  {
    "id": "1",
    "title": "Do something",
    "content": "A description about this task.",
    "is_completed": false
  }
}
/PUT /api/todos/{id}

Description

Modify an existing todo by ID.

Content Type

application/json

Request Body

{
  "title": "Your new title",
  "content": "Your new description",
  "is_completed": true
}

Sample Response

{
  "message": "ok",
  "todo":
    {
      "id": "1",
      "title": "Your new title",
      "content": "Your new description",
      "is_completed": true
    }
  }
